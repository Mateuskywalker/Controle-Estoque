Instruções para que o Banco de dados funcione corretamente:

        * Para que o banco de dados (sql) funcione corretamente é necessário ter o XAMPP instalado e
        com os seguintes itens rodando:

            server            ---------------- status

            MySQL Database                      running
            Apache Web Server                   running


        ** O passe seguinte é importar a tabela "NOTAS" ao banco:

            - acesse: http://localhost/phpmyadmin/.

            - Crie um banco de dados chamado: controle.

            - Com esse mesmo banco selecionado, vá até a parte superior direita do monitor e clique em
             importar.

            - Importe as três tabelas sql que estarão no mesmo diretório da aplicação(tabela:'Cliente', 'Pedido' e 'Produto' dentro da pasta
            banco-de-dados)

Instruções para rodar a aplicação:

        * Vá até a pasta raiz do seu computador, onde estará pasta do XAMPP e coloque a pasta descompactada que enviei dentro da pasta htdocs.

        * Abra o navegador e digite: http://localhost/master-controle-abril/index.php

        * A aplicação então estará rodando.
